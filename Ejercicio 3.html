Las 5 aplicaciones web más famosas en la actualidad

Facebook

Fecha de lanzamiento de la app: Octubre 2004

Empresa encargada de su desarrollo: Meta

Descripción

Facebook es una plataforma web gratuita de redes sociales que promueve y facilita la interacción entre amigos, familiares y compañeros de trabajo. Facebook fue fundada en 2004 por Mark Zuckerberg y varios compañeros de la Universidad de Harvard, hoy la empresa se llama Meta.

Lista de tecnologías/herramientas implementadas en su desarrollo.

  • Java
  • C++
  • Erlang
  • Python
  • Haskell
  • PHP

WhatsApp

Fecha de lanzamiento de la app: 2009

Empresa encargada de su desarrollo: Meta

Descripción

Es una aplicación de mensajería instantánea para smartphones. La aplicación nos permite enviar no solo textos sino también imágenes, videos y audios

Lista de tecnologías/herramientas implementadas en su desarrollo.

  • Java Script
  • HTML/CSS
  • SQL
  • Python
  • Type Script

YouTube

Fecha de lanzamiento de la app: Febrero 2005

Empresa encargada de su desarrollo: Google LLC

Descripción

YouTube es un sitio web de origen estadounidense dedicado a compartir videos. Presenta una variedad de clips de películas, programas de televisión y vídeos musicales, así como contenidos amateur como videoblogs. Las personas que crean contenido para esta plataforma generalmente son conocidas como youtubers

Lista de tecnologías/herramientas implementadas en su desarrollo.

  • TubeBuddy
  • VidIQ
  • MorningFa.me
  • Keyword Keg
  • Kparser

Messenger

Fecha de lanzamiento de la app: 2008

Empresa encargada de su desarrollo: Meta

Descripción

Messenger permite a los usuarios realizar llamadas de voz y videollamadas tanto en interacciones individuales como en conversaciones de grupo.

Lista de tecnologías/herramientas implementadas en su desarrollo.

  • Java
  • C++
  • Erlang
  • Python
  • Haskell
  • PHP

WeChat

Fecha de lanzamiento de la app: 2011

Empresa encargada de su desarrollo: Tencent Holdings

Descripción

WeChat provee un servicio de mensajería que incluye mensajes de texto, mensajes de voz, transmisiones en vivo (de un usuario a múltiples usuarios), videollamadas y llamadas de conferencia, envío de imágenes, vídeos y videojuegos.

Lista de tecnologías/herramientas implementadas en su desarrollo.

  • Java
  • C++
  • Erlang
  • Python
